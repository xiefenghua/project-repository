### 第三方短信发送模块引入说明

#### 1. 引入依赖

```xml
        <dependency>
            <groupId>com.security.im</groupId>
            <artifactId>xxqa-common-sms</artifactId>
            <version>1.0-SNAPSHOT</version>
        </dependency>
```

#### 2. 增加短信模板配置
```xml
sms.template.authCode=6666
```
